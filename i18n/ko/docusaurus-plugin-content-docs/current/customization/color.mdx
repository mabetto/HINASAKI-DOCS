# 색상 선택 방법

알아두면 완성도가 달라지는 색상 선택 팁을 정리하였습니다.

<ColorPalette />

---

## 70:25:5 법칙

배색의 기본 비율로 널리 사용됩니다.

| 역할 | 비율 | 예시 |
|------|------|-----|
| 베이스 컬러 | 70% | 헤어·메인 의상 색상 |
| 서브 컬러 | 25% | 의상 서브 파츠·액센트 소재 |
| 액센트 컬러 | 5% | 버튼·보석·포인트 색상 |

<TipsCard>액센트 컬러는 면적이 작기 때문에 오히려 눈에 잘 띕니다. 강한 주장의 색상은 여기에 집중하는 것이 효과적입니다.</TipsCard>

## 원색은 피하기

RGB 값이 극단적인 색상（순수한 빨강 `255, 0, 0` 등）은 VRChat의 라이팅 환경이나 월드의 포스트 이펙트에 강한 영향을 받습니다. 밝은 월드에서는 채도가 포화되어 뭉개지고, 어두운 월드에서는 부자연스럽게 떠 보이는 경우가 많습니다.

<TipsCard>채도를 조금 낮추고 명도를 조정하는 것만으로도 어우러짐이 크게 개선됩니다.</TipsCard>

---

## 색의 3속성

색은 **색상·채도·명도** 3가지 요소로 구성됩니다. 대부분의 페인트 소프트웨어와 Unity의 컬러 피커에서 HSV/HSL 모드로 조작할 수 있습니다.

| 속성 | 의미 |
|------|------|
| 색상（Hue） | 빨강·파랑·초록 등 「무슨 색인지」를 나타냅니다. 색상환 위의 위치로 표시됩니다. |
| 채도（Saturation） | 색의 선명함. 높을수록 원색에 가깝고, 낮을수록 회색에 가까워집니다. |
| 명도（Value / Brightness） | 색의 밝기. 높을수록 흰색에, 낮을수록 검정에 가까워집니다. |

## 색상 선택 방법

### 유사색으로 통일하기

인접한 색상（예：파랑 → 청보라 → 보라）으로 통일하면 자연스럽게 조화를 이룹니다.

<TipsCard>초보자도 실패하기 어려운 배색입니다. 먼저 유사색으로 통일하고, 익숙해지면 보색을 소량 추가하는 것을 권장합니다.</TipsCard>

### 보색으로 액센트 넣기

색상환의 반대쪽 색상（예：파랑에 대한 오렌지）을 소량 사용하면 시선을 집중시킬 수 있습니다.

<TipsCard>액센트 컬러에 보색을 사용하는 것이 정석입니다. 위의 팔레트 툴에서 「보색」항목을 참고하십시오.</TipsCard>

### 색 온도 통일하기

난색계（빨강·주황·노랑）와 한색계（파랑·청록）가 혼재하면 인상이 통일되기 어렵습니다.

<TipsCard>전체를 어느 한쪽 색 온도로 맞추고 액센트에 반대쪽을 사용하는 것이 다루기 쉬운 구성입니다.</TipsCard>

## 채도·명도 컨트롤

### 채도 통일하기

베이스 컬러와 서브 컬러의 채도 레벨이 크게 차이나면 파츠별로 「떠 보이는」인상을 줍니다.

<TipsCard>기존 텍스처의 채도를 참고하여 추가 머티리얼도 동일한 범위에 맞추면 통일감이 생깁니다.</TipsCard>

### 명도 차이로 메리하리 주기

같은 색상이라도 명도를 바꾸면 입체감과 무게중심이 생깁니다.

<TipsCard>상반신을 밝게, 하반신을 약간 어둡게 하는 것만으로도 실루엣이 引き締まります.</TipsCard>

## 라이팅 설정을 전제로 색상 결정하기

3+1은 `_LightMinLimit: 0.05`에 의해 어두운 월드에서도 최소한의 명도가 유지되고, `_MonochromeLighting: 0.5`에 의해 유채색 라이트가 절반 회색으로 보정됩니다.

<TipsCard>자신의 제작 환경（Directional Light 하나）만으로 색상을 결정하면 VRChat 내에서 예상과 다른 결과가 나오기 쉬우므로 여러 라이팅 환경에서 확인하는 것을 권장합니다.</TipsCard>

## 원본 텍스처의 색역을 참고하기

새로운 의상이나 액세서리 텍스처를 제작할 때는 원본 `SA_CLOTH_A.psd`나 `SA_HAIR_A.psd`의 색역（채도·명도 범위）을 참고하면 통일감이 생깁니다.

<TipsCard>원본 텍스처에서 스포이트로 색을 추출하여 거기서 파생시키는 방법이 가장 자연스럽게 어우러집니다.</TipsCard>

## 개선 포인트

| 상황 | 개선 방향 |
|------------|------------|
| 색이 너무 많음 | 색상을 3~4종류로 제한하고 명도 차이로 변화를 줌 |
| 전체 채도가 동일함 | 베이스를 저채도로 하고 액센트만 고채도로 함 |
| 원색 남용 | 순색에 흰색·검정·회색을 조금 섞으면 자연스러워짐 |
| 대비가 약함 | 명도 차이를 의식적으로 만들면 읽기 쉬운 실루엣이 됨 |
