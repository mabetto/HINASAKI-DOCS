## FX 컨트롤

왼손·오른손 각각의 제스처 변경 버튼을 전환함으로써, 표준 8포즈 외에 고양이·하트·손가락 하트·건·전화 등의 추가 핸드 제스처를 사용할 수 있습니다. 좌우를 독립적으로 설정할 수 있어 양손으로 서로 다른 제스처를 동시에 사용하는 것도 가능합니다.

## 눕기
