## System Requirements

| Item | Recommended |
|------|------|
| Unity | 2022.3.22f1 LTS |
| VRChat SDK | VRChat SDK3A (Avatar 3.0) |
| Shader | lilToon |
| Physics | PhysBone (DynamicBone not supported) |
| Modular Avatar | Recommended (native support) |
| VRCFury | Required when using SE extension |
