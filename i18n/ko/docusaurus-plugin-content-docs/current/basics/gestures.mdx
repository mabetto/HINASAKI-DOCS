# 제스처

왼손과 오른손의 제스처는 독립적으로 제어할 수 있습니다.

---

## 제스처 대응표

**FX 컨트롤**의 **왼손 제스처 변경**（`G_L_Ctrl`）/ **오른손 제스처 변경**（`G_R_Ctrl`）을 ON으로 하면, 동일한 제스처 입력으로 대체 포즈가 출력됩니다. 좌우는 독립적으로 설정할 수 있습니다.

{/* ###image### */}

| 값 | 표준（Ctrl OFF） | 추가（Ctrl ON） |
|----|-----------------|----------------|
| 0 | Idle | — |
| 1 | ✊ 주먹 | ✊ 주먹 |
| 2 | 🖐️ 펼친 손 | 🖐️ 펼친 손 |
| 3 | 🫵 손가락 지목 | 🫶 하트 |
| 4 | ✌️ 브이 | 🫰 손가락 하트 |
| 5 | 🤘 뿔 | 🖕 가운데 손가락 |
| 6 | <span style={{display:'inline-block',transform:'rotate(90deg)'}}>👆</span> 건 | 🤙 전화 |
| 7 | 👍 엄지 올리기 | 👌 OK / 돈 |

---

## 핸드 트래킹

상단 메뉴의 **핸드 트래킹**을 ON으로 하면 Index 컨트롤러의 핑거 트래킹이 활성화됩니다（`Index` 파라미터）.
