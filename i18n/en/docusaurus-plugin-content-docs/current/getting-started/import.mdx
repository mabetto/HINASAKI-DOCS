# Importing into Unity

:::note[Note]

An avatar project must already be created in VRChat Creator Companion (VCC).

:::

---

## Steps

1. Add the following repository URL to VCC and install **Modular Avatar**.

   ```
   vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm.json
   ```

2. Import the latest version of **lilToon** into the `Assets` folder.

3. Import `3+1_PKG.unitypackage` into the `Assets` folder.

4. Open `Assets/HINASAKI/3+1/3+1_Upload.unity`.

5. From the Unity menu, open **VRChat SDK → Show Control Panel**.

6. Log in with your VRChat account.

7. Enter the avatar name in the **Builder** tab.

8. Click **Build & Publish** to upload.

---

## Installing Language Extensions

The language of the expressions menu can be switched. The default language is Japanese.

| File | Language |
|---------|------|
| `3+1_LANG_EN_EXT.unitypackage` | English |
| `3+1_LANG_KO_EXT.unitypackage` | 한국어 |

1. Complete steps 1–4 above.
2. Import the language extension `.unitypackage` into the `Assets` folder to overwrite the expressions menu.
3. Perform steps 5–8 to upload.
