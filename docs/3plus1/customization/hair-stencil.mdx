# 前髪のステンシル設定

３＋１では、前髪が顔に重なっても**眉・アイラインが透けて見える**よう、ステンシルバッファを使った髪抜け処理を行っています。新しい前髪を追加する場合は、同じ設定をマテリアルに行う必要があります。設定しないと眉・アイラインが前髪に隠れてしまいます。

仕組みの詳細は [シェーダー設定 › ステンシルシステム](/3plus1/settings/#ステンシルシステム) を参照してください。

---

## 設定手順

### 1. 前髪のマテリアルを選択する

Project ウィンドウまたは Hierarchy から前髪オブジェクトを選択し、使用しているマテリアルを Inspector で開きます。

### 2. lilToon のステンシル設定を開く

lilToon のマテリアルインスペクターで **「ステンシル」** セクションを展開します。

<TipsCard>セクションが見つからない場合は、インスペクター上部の表示モードを「中級者」以上に切り替えてください。</TipsCard>

### 3. 値を設定する

| 項目 | 値 |
|------|-----|
| Stencil Ref | **1** |
| Stencil Comp | **NotEqual** |
| Stencil Pass | Keep |
| Stencil Fail | Keep |
| Stencil ZFail | Keep |

<TipsCard>Pass / Fail / ZFail はデフォルトの Keep のまま変更不要です。Ref と Comp の2項目だけ変更してください。</TipsCard>

### 4. 複数マテリアルがある場合

前髪オブジェクトに複数のマテリアルが割り当てられている場合は、**すべてのマテリアル**に同じ設定を行ってください。

### 5. Play モードで確認する

Unity の Play モードに入り、眉・アイラインが前髪越しに見えているか確認してください。

---

## lilToon の設定を合わせる

プリファレンスメニューの **明度**（`Brightness` パラメーター）と **ライトのモノクロ化**（`Lighting(mono)` パラメーター）は、COS_A のアニメーターが各マテリアルの `_LightMinLimit` / `_MonochromeLighting` を直接操作することで機能しています。

新しく追加したメッシュのマテリアルはこのアニメーターの対象外になるため、**デフォルト値を本体と揃えておく**ことで、ラジアル未操作時の見た目を統一できます。

詳細は [シェーダー設定](/3plus1/settings/) を参照してください。

| プロパティ | 最小値（ラジアル 0） | 最大値（ラジアル 1） |
|-----------|------|------|
| `_LightMinLimit` | 0.05 | 1.0 |
| `_MonochromeLighting` | 0.5 | 1.0 |

<TipsCard>ラジアルで明度やモノクロを動かしたときに新規パーツだけ追従させたい場合は、COS_A のアニメーションクリップに新規メッシュのマテリアルスロットを追記する必要があります。</TipsCard>

## チェックリスト

- [ ] 前髪マテリアルの `Stencil Ref` が **1** になっている
- [ ] 前髪マテリアルの `Stencil Comp` が **NotEqual** になっている
- [ ] 複数マテリアルがある場合はすべてに設定済み
- [ ] Play モードで眉・アイラインが透けて見える
- [ ] lilToon のライティング設定値が本体と揃っている
