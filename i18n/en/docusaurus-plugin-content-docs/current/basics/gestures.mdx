# Gestures

Left and right hand gestures can be controlled independently.

---

## Gesture Reference Table

Turning on **Left Hand Gesture Change** (`G_L_Ctrl`) / **Right Hand Gesture Change** (`G_R_Ctrl`) in **FX Controls** produces alternative poses for the same gesture input. Left and right can be set independently.

{/* ###image### */}

| Value | Standard (Ctrl OFF) | Additional (Ctrl ON) |
|----|-----------------|----------------|
| 0 | Idle | — |
| 1 | ✊ Fist | ✊ Fist |
| 2 | 🖐️ Open hand | 🖐️ Open hand |
| 3 | 🫵 Point | 🫶 Heart |
| 4 | ✌️ Peace | 🫰 Finger heart |
| 5 | 🤘 Horns | 🖕 Middle finger |
| 6 | <span style={{display:'inline-block',transform:'rotate(90deg)'}}>👆</span> Gun | 🤙 Phone |
| 7 | 👍 Thumbs up | 👌 OK / Money |

---

## Hand Tracking

Turning on **Hand Tracking** in the top menu enables finger tracking for the Index controller (`Index` parameter).
