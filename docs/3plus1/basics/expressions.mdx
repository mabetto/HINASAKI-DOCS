# 表情

アクションメニューの **FXコントロール** から表情パレットを切り替えられます。

---

## 表情パレット

表情は3つのセットに分かれています。**FXコントロール** でセットA / B / C を切り替えます（`Facial_palette`）。各セットの表情は左手のジェスチャーで切り替えます。

## MMD互換モード

**互換モード.app** を ON にすると MMD モーション等との互換表情に切り替わります（`MMD` パラメーター）。
