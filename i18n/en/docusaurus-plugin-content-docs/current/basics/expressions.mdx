# Expressions

The expression palette can be switched from **FX Controls** in the action menu.

---

## Expression Palette

Expressions are divided into three sets. Switch between Set A / B / C in **FX Controls** (`Facial_palette`). Within each set, expressions are switched using left-hand gestures.

## MMD Compatibility Mode

Turning on **MMD.app** switches to compatibility expressions for MMD motions etc. (`MMD` parameter).
