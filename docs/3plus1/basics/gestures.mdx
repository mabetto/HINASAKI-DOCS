# ジェスチャー

左右の手のジェスチャーは独立して制御できます。

---

## ジェスチャー対応表

**FXコントロール** の **左手ジェスチャー変更**（`G_L_Ctrl`）/ **右手ジェスチャー変更**（`G_R_Ctrl`）を ON にすると、同じジェスチャー入力で代替ポーズが出ます。左右は独立して設定できます。

| 値 | 標準（Ctrl OFF） | 追加（Ctrl ON） |
|----|-----------------|----------------|
| 0 | Idle | — |
| 1 | ✊ グー | ✊ グー |
| 2 | 🖐️ パー | 🖐️ パー |
| 3 | 🫵 指差し | 🫶 ハート |
| 4 | ✌️ チョキ | 🫰 ゆびハート |
| 5 | 🤘 きつね | 🖕 中指立て |

| 7 | 👍 サムズアップ | 👌 OK / お金 |

---

## ハンドトラッキング

トップメニューの **ハンドトラッキング** を ON にすると、Index コントローラーのフィンガートラッキングが有効になります（`Index` パラメーター）。
