# 표정

액션 메뉴의 **FX 컨트롤**에서 표정 팔레트를 전환할 수 있습니다.

---

## 표정 팔레트

표정은 3개의 세트로 나뉩니다. **FX 컨트롤**에서 세트 A / B / C를 전환합니다（`Facial_palette`）. 각 세트의 표정은 왼손 제스처로 전환합니다.
