# SE 확장 설치

SE 확장은 무료 DLC입니다. 본체와 별도로 다운로드·설치가 필요합니다.

---

## 순서

1. VCC에 다음 URL로 리포지토리를 추가하고 **Modular Avatar**를 설치합니다.

   ```
   vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm.json
   ```

2. VCC에 다음 URL로 리포지토리를 추가하고 **VRCFury**를 설치합니다.

   ```
   vcc://vpm/addRepo?url=https%3A%2F%2Fvcc.vrcfury.com
   ```

3. **lilToon** 최신 버전을 `Assets` 폴더에 불러옵니다.

4. `3+1_PKG.unitypackage`와 `3+1_SE_EXT.unitypackage` 두 파일 모두 `Assets` 폴더에 불러옵니다.

5. `Assets/HINASAKI/3+1/3+1_Upload.unity`를 엽니다.

6. `Assets/HINASAKI/A2XL_SE_Extension/Prefab/`에 있는 **SA_A2XL_R18** 프리팹을 Hierarchy의 **3+1**에 드래그 앤 드롭하여 배치합니다.

7. `SA_A2XL_R18` 안의 **Basebody** 셰이프 키를 의상에 맞게 설정합니다.

   초기 의상（COS_A）값：

   | 셰이프 키 | 값 |
   |------------|-----|
   | `adj_bust_cloth1` | 100 |
   | `adj_pumps` | 100 |
   | `opt_hekochiku` | 10 |

:::info[주의]

**3+1 Large 배리언트의 경우에만：** `SA_A2XL_R18`의 `Armature`에 연결된 **MA Merge Armature**에서 "위치를 원래 아바타에 맞춰 리셋" → "스케일도 맞추기"에 체크하고 실행합니다.

:::

8. Unity 메뉴에서 **VRChat SDK → Show Control Panel**을 열고 업로드합니다.

---

## 언어 확장 설치

SE 확장의 익스프레션 메뉴도 일본어·영어·한국어를 지원합니다.

| 파일 | 언어 |
|---------|------|
| `3+1_SE_LANG_EN_EXT.unitypackage` | English |
| `3+1_SE_LANG_KO_EXT.unitypackage` | 한국어 |

1. 위 순서 1~6을 완료합니다.
2. 언어 확장 `.unitypackage`를 `Assets` 폴더에 불러옵니다.
3. 순서 8을 실행하여 업로드합니다.
