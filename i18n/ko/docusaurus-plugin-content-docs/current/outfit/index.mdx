# MA 설정

3+1은 **Modular Avatar**를 전제로 한 비파괴 구성입니다. 의상 및 액세서리 추가는 모두 MA로 수행하며, 베이스 아바타 데이터는 직접 편집하지 않습니다.

---

## COS_A / COS_B 구성

```
3+1（베이스 아바타）
├─ SA_COS_A
└─ SA_COS_B
```

COS_A와 COS_B는 각각 독립된 프리팹으로 Hierarchy에 배치되며, MA Merge Animator가 각자의 FX Controller를 베이스 아바타의 FX 레이어에 머지합니다. 두 프리팹은 항상 존재하며, 표시·비표시 전환은 애니메이터가 제어합니다.

COS_B는 COS_A의 '캐스트오프 후 상태'를 베이스로 합니다. COS_A의 `CostumeBody` 파라미터가 OFF가 되면 COS_A의 의상 메시가 비표시가 되고, 대신 COS_B의 메시가 표시되는 구조입니다. COS_B 전용 FX Controller는 주로 바디 변형과 라이팅 설정만 담당하며, 옷장이나 램프 등의 메뉴는 갖지 않습니다.

| | COS_A | COS_B |
|--|-------|-------|
| 스타일 | 교복 | 나체（COS_A 캐스트오프 후의 베이스） |
| FX Controller | 전용 레이어를 베이스에 머지 | 전용 레이어를 베이스에 머지 |
| 메뉴 추가 | 옷장 · 램프 | 없음 |

## 컴포넌트 구성

새 코스튬 프리팹을 추가할 때는 다음 MA 컴포넌트를 조합하여 구성합니다.

| 컴포넌트 | 역할 |
|--------------|------|
| MA Merge Animator | 코스튬용 FX Controller를 베이스에 머지 |
| MA Parameters | 코스튬 고유 파라미터를 추가 |
| MA Menu Installer | 의상 메뉴를 최상위 메뉴에 통합 |
| MA Bone Proxy | 베이스 아바타의 본을 따라감 |

MA Merge Animator는 레이어를 비파괴적으로 머지하므로 기존 FX 레이어를 덮어쓰지 않습니다. `matchAvatarWriteDefaults: true`를 통해 베이스 아바타의 Write Defaults 설정과 자동으로 통일됩니다.

MA Merge Animator 설정값：

| 설정 | 값 |
|------|-----|
| layerType | 5（FX） |
| pathMode | 0（Relative） |
| matchAvatarWriteDefaults | true |
| deleteAttachedAnimator | true |

## 파라미터 목록

### SA_COS_A FX

COS_A의 FX Controller가 참조하는 파라미터입니다. `GestureLeft` / `GestureRight` / `Facial_palette`는 베이스 아바타의 FX Controller（표정 · 제스처 레이어）와 공유됩니다. `Brightness` / `Lighting(mono)`도 마찬가지로 베이스 아바타와 공유하며, COS_A 전용 애니메이션 클립으로 각 머티리얼의 밝기를 제어합니다.

`CostumeHead` / `CostumeBody` / `CostumeSkirt`는 COS_A 고유의 파라미터로, 옷장 메뉴의 각 버튼에 대응합니다. `CostumeHead`와 `CostumeBody`가 Int 타입인 것은 의상이 늘어날 때 대응할 수 있도록 여유를 확보한 설계입니다. `Turnsignal`은 값에 따라 방향지시등의 동작 모드가 바뀝니다.

| 파라미터 | 타입 | 설명 |
|------------|---|------|
| `GestureLeft` | Int | VRChat 내장（왼손 제스처） |
| `GestureRight` | Int | VRChat 내장（오른손 제스처） |
| `Facial_palette` | Int | 표정 세트 선택（0=A / 1=B / 2=C） |
| `CostumeHead` | Int | 머리 의상 표시 전환 |
| `CostumeBody` | Int | 바디 의상 표시 전환 |
| `CostumeSkirt` | Bool | 스커트 표시 전환 |
| `Daylight` | Bool | 데이라이트 ON/OFF |
| `Turnsignal` | Int | 방향지시등（0=OFF / 1=비상등 / 2=좌 / 3=우） |
| `Brightness` | Float | 밝기 조정 |
| `Lighting(mono)` | Float | 라이트 모노크롬화 |

### SA_COS_B FX

COS_B의 FX Controller는 단순한 구성으로, `CostumeBody`에 의한 바디 변형 전환과 라이팅 설정만을 담당합니다. `Brightness` / `Lighting(mono)`는 베이스 아바타와 공유되며, COS_B 전용 애니메이션 클립으로 COS_B의 머티리얼에 적용됩니다.

| 파라미터 | 타입 | 설명 |
|------------|---|------|
| `CostumeBody` | Int | 바디 의상 표시 전환 |
| `Brightness` | Float | 밝기 조정 |
| `Lighting(mono)` | Float | 라이트 모노크롬화 |
