# ごろね

トップメニューの **ごろね** からポーズを選択できます。

:::note[注記]

ごろねは VRChat 標準の「しゃがみ・伏せ」動作を**置換する**システムです。メニューでポーズを選択しただけでは発動しません。選択後に **HMD を物理的に下げる**（しゃがむ・伏せる）か、**OVR Advanced Settings の Spacemover** でプレイスペースを Drag することで発動します。

:::

---

- うつ伏せ
- 仰向け
- 横向き
- あおむけ / ぺたん
- **ポーズ遷移ロック** — ポーズが変化しないようにロック（`VRCLockPose`）
- **フットアンカー** — 足の位置を床に固定（`VRCFootAnchor`）

:::note[注記]

ごろね中に **ポーズ遷移ロック** を ON にすると、移動やアクションによってポーズが解除されなくなります。

:::
