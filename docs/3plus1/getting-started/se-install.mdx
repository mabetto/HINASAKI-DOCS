# SE拡張のインストール

SE拡張は無料DLCです。本体とは別にダウンロード・インストールが必要です。

---

## 手順

1. VCC に以下の URL からリポジトリを追加し、**Modular Avatar** をインストールします。

   ```
   vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm.json
   ```

2. VCC に以下の URL からリポジトリを追加し、**VRCFury** をインストールします。

   ```
   vcc://vpm/addRepo?url=https%3A%2F%2Fvcc.vrcfury.com
   ```

3. **lilToon** の最新版を `Assets` フォルダに読み込みます。

4. `3+1_PKG.unitypackage` と `3+1_SE_EXT.unitypackage` を両方 `Assets` フォルダに読み込みます。

5. `Assets/HINASAKI/3+1/3+1_Upload.unity` を開きます。

6. `Assets/HINASAKI/A2XL_SE_Extension/Prefab/` にある **SA_A2XL_R18** プレハブを、Hierarchy 内の **3+1** にドラッグ＆ドロップして配置します。

7. `SA_A2XL_R18` 内の **Basebody** のシェイプキーを衣装に合わせて設定します。

   初期衣装（COS_A）の値：

   | シェイプキー | 値 |
   |------------|-----|
   | `adj_bust_cloth1` | 100 |
   | `adj_pumps` | 100 |
   | `opt_hekochiku` | 10 |

:::info[注意]

**3+1 Large バリアントの場合のみ：**`SA_A2XL_R18` の `Armature` にアタッチされている **MA Merge Armature** の「位置を元アバターに合わせてリセット」→「スケールも合わせる」にチェックを入れて実行します。

:::

8. Unity メニューから **VRChat SDK → Show Control Panel** を開き、アップロードします。

---

## 言語拡張のインストール

SE拡張のエクスプレッションメニューも日本語・英語・韓国語に対応しています。

| ファイル | 言語 |
|---------|------|
| `3+1_SE_LANG_EN_EXT.unitypackage` | English |
| `3+1_SE_LANG_KO_EXT.unitypackage` | 한국어 |

1. 上記の手順 1〜6 を完了させます。
2. 言語拡張の `.unitypackage` を `Assets` フォルダに読み込みます。
3. 手順 8 を実行してアップロードします。
