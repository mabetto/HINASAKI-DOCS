# 트러블슈팅 · FAQ

<FaqCard q="SE 확장 아카이브를 압축 해제하는 비밀번호를 모르겠습니다">

본체 패키지（`3+1_PKG.unitypackage`）안에 동봉된 `PASSWORD.txt`에 기재되어 있습니다. 먼저 본체 패키지를 임포트한 후 확인해 주십시오.

</FaqCard>

<FaqCard q="셰이더가 핑크색이 됩니다">

lilToon 셰이더가 임포트되어 있지 않습니다. [lilToon 배포 페이지](https://lilxyzw.github.io/lilToon/)에서 임포트해 주십시오.

</FaqCard>

<FaqCard q="NDMF / Modular Avatar 오류가 발생합니다">

VCC（VRChat Creator Companion）에서 최신 Modular Avatar를 설치해 주십시오.

</FaqCard>

<FaqCard q="AvatarPoseSystem 사용 시 물리 오브젝트가 변형됩니다">

AvatarPoseSystem（APS）의 PhysBone 고정 기능이 3+1의 모든 물리 오브젝트 PhysBone과 간섭하여, 물리 오브젝트가 의도치 않은 형태로 변형된 채 돌아오지 않는 경우가 있습니다.

APS를 도입할 때는 사용하는 모든 PhysBone을 **Unfix**로 설정해 주십시오. 또는 APS를 사용하지 않는 환경에서 촬영해 주십시오.

</FaqCard>

<FaqCard q="추가한 앞머리 파츠로 인해 눈썹이 가려집니다">

추가하는 앞머리 머티리얼에 `StencilComp: NotEqual / StencilRef: 1` 설정이 필요합니다. 자세한 내용은 [셰이더 설정](/3plus1/settings/)을 참조해 주십시오.

</FaqCard>

<FaqCard q="의상을 전환하면 셰이프 키 값（adj_bust_cloth1 등）이 리셋됩니다">

COS_A는 **MA Shape Changer**로 바디 메시의 셰이프 키를 제어하고 있습니다. 의상을 OFF로 하면 MA Shape Changer가 해제되어 제어하던 셰이프 키 값이 0으로 돌아갑니다. 이는 MA Shape Changer의 사양입니다.

다른 의상에서도 같은 셰이프 키를 유지하고 싶은 경우, 해당 의상 오브젝트에도 **MA Shape Changer**를 추가하여 같은 셰이프 키와 값을 설정해 주십시오. 자세한 내용은 [헤어·의상 추가（MA）](/3plus1/customization/ma-outfit)를 참조해 주십시오.

</FaqCard>

<FaqCard q="추가한 머티리얼만 밝기 조정이 적용되지 않습니다">

`_MonochromeLighting`의 초기값을 `0.5`로 설정해 주십시오. 자세한 내용은 [셰이더 설정](/3plus1/settings/)을 참조해 주십시오.

</FaqCard>
