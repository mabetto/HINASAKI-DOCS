# 앞머리 스텐실 설정

3+1에서는 앞머리가 얼굴에 겹쳐도 **눈썹·아이라인이 비쳐 보이도록** 스텐실 버퍼를 사용한 헤어 투과 처리를 적용하고 있습니다. 새로운 앞머리를 추가하는 경우에는 동일한 설정을 머티리얼에 적용해야 합니다. 설정하지 않으면 눈썹·아이라인이 앞머리에 가려집니다.

이 기능의 원리에 대한 자세한 내용은 [셰이더 설정 › 스텐실 시스템](/3plus1/settings/#스텐실-시스템)을 참조하십시오.

---

## 설정 절차

### 1. 앞머리 머티리얼 선택

Project 창 또는 Hierarchy에서 앞머리 오브젝트를 선택하고, 사용 중인 머티리얼을 Inspector에서 엽니다.

### 2. lilToon의 스텐실 설정 열기

lilToon 머티리얼 Inspector에서 **「스텐실」** 섹션을 펼칩니다.

<TipsCard>섹션이 보이지 않는 경우 Inspector 상단의 표시 모드를 「중급자」이상으로 전환하십시오.</TipsCard>

### 3. 값 설정

| 항목 | 값 |
|------|-----|
| Stencil Ref | **1** |
| Stencil Comp | **NotEqual** |
| Stencil Pass | Keep |
| Stencil Fail | Keep |
| Stencil ZFail | Keep |

<TipsCard>Pass / Fail / ZFail은 기본값인 Keep 그대로 변경하지 않아도 됩니다. Ref와 Comp 두 항목만 변경하십시오.</TipsCard>

### 4. 머티리얼이 여러 개인 경우

앞머리 오브젝트에 머티리얼이 여러 개 할당되어 있는 경우에는 **모든 머티리얼**에 동일한 설정을 적용하십시오.

### 5. Play 모드에서 확인

Unity의 Play 모드에서 눈썹·아이라인이 앞머리 너머로 보이는지 확인하십시오.

---

## lilToon 설정 맞추기

프리퍼런스 메뉴의 **밝기**（`Brightness` 파라미터）와 **라이트 모노크로화**（`Lighting(mono)` 파라미터）는 COS_A의 애니메이터가 각 머티리얼의 `_LightMinLimit` / `_MonochromeLighting`을 직접 조작함으로써 기능합니다.

새로 추가한 메시의 머티리얼은 이 애니메이터의 대상에서 제외되므로, **기본값을 본체와 맞춰 두면** 라디얼 미조작 시 외관을 통일할 수 있습니다.

자세한 내용은 [셰이더 설정](/3plus1/settings/)을 참조하십시오.

| 프로퍼티 | 최솟값（라디얼 0） | 최댓값（라디얼 1） |
|-----------|------|------|
| `_LightMinLimit` | 0.05 | 1.0 |
| `_MonochromeLighting` | 0.5 | 1.0 |

<TipsCard>라디얼로 밝기나 모노크로를 조작할 때 새 파츠만 따라오게 하려면 COS_A의 애니메이션 클립에 새 메시의 머티리얼 슬롯을 추가해야 합니다.</TipsCard>

---

## 체크리스트

- [ ] 앞머리 머티리얼의 `Stencil Ref`가 **1**로 설정되어 있음
- [ ] 앞머리 머티리얼의 `Stencil Comp`가 **NotEqual**로 설정되어 있음
- [ ] 머티리얼이 여러 개인 경우 모두 설정 완료
- [ ] Play 모드에서 눈썹·아이라인이 비쳐 보임
- [ ] lilToon 라이팅 설정값이 본체와 맞춰져 있음
